# GitHub für Entwickler – CI/CD Demo Repository

Dieses Repository gehört zur Schulung **„GitHub für Entwickler“** (GFU, 24.–25.11.2025).

Es enthält **sprache- und technologieunabhängige Beispiele**, die zeigen:

- wie CI-Pipelines auf GitHub Actions funktionieren  
- wie Build-Artefakte erstellt werden  
- wie Release-Artefakte versioniert und ausgeliefert werden  
- wie GitFlow in Kombination mit CI/CD angewendet wird  
- wie man PowerShell-basierte Tests ausführt (neutral für PHP, C#, JS, Python etc.)

**Wichtig:**  
Dieses Repository enthält **keinen produktiven Code**, sondern reine Demonstrations- und Schulungsartefakte für CI/CD und Software-Paketierung.

## Inhalt

- `src/` – Beispielquellmaterial (codeneutral)
- `test.ps1` – generisches Testszenario für CI
- `.github/workflows/*` – CI, Build und Release Pipelines
- `docs/` – weiterführende Erklärungen
